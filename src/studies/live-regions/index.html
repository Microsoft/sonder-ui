Disaster management dashboard

Apologies to anyone who is familiar with disaster management, because this is not going to be at all realistic. The purpose of this study is to look at three different types of live region announcements:

  • Loading announcements
  • Filtering/other input announcements
  • Real-time data announcements
  • Houston (least verbose)
  • Galveston (middle verbosity)
  • New Orleans (most verbose)
